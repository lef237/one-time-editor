# One-Time Editor

A temporary notepad app for drafting chat messages and prompts before sending. Built with Electron, React, and TypeScript.

## Features

- **Editor** — Monospace text area for drafting
- **New** — Saves current text to history and clears the editor
- **Copy** — Copies text to clipboard and saves to history
- **History** — Browse and restore past entries (up to 100)
- **Global Shortcut** — Toggle the window with `Ctrl+M` (or `Cmd+M` on macOS), configurable in settings
- **Auto-hide** — Window hides when it loses focus

## Setup

```bash
npm install
npm run dev
```
